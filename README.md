# Генератор документов для частных охранных организаций

Десктопное приложение для автоматизации подготовки кадровых документов в частных охранных организациях (ЧОП).

Программа позволяет загружать данные сотрудников из Excel, автоматически подставлять их в готовые шаблоны документов и значительно сокращать время на оформление кадровой документации.

## Возможности

- Работа с несколькими организациями и быстрое переключение между ними.
- Импорт списка сотрудников из файлов Excel (`.xlsx`).
- Удобная таблица сотрудников с поиском и сортировкой.
- Выбор одного или сразу нескольких сотрудников для формирования документов.
- Автоматическое заполнение документов по готовым шаблонам.
- Генерация различных кадровых документов, в том числе:
  - заявление на получение личной карточки;
  - уведомление о приеме на работу;
  - уведомление об увольнении;
  - и другие документы.
- Автоматическое сохранение готовых документов на рабочий стол.
- Сохранение настроек и данных между запусками приложения.

## Как пользоваться

1. Запустите приложение.
2. Выберите необходимую организацию.
3. Нажмите кнопку **«Обновить»** и загрузите Excel-файл с данными сотрудников.
4. Найдите нужного сотрудника через поиск или выберите нескольких сотрудников из списка.
5. Нажмите **«Сгенерировать документы»**.
6. В открывшемся окне выберите тип документа.
7. После завершения генерации готовый документ автоматически сохранится на рабочем столе.

## Назначение

Приложение разработано для упрощения повседневной работы сотрудников кадровых служб частных охранных организаций. Оно избавляет от необходимости вручную заполнять однотипные документы, снижает вероятность ошибок и позволяет подготовить необходимый пакет документов всего за несколько кликов.
